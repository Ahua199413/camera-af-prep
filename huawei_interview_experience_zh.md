# 華為日本品川研究所 - 相機演算法團隊面試經驗與流程分享

本指南整理了社群（PTT Oversea_Job、Dcard、知乎）上關於**華為日本研究所（品川/橫濱）**研發職缺與相機/演算法團隊的真實面試經驗、關卡細節及準備建議。

---

## 1. 招募與面試流程全貌 (Interview Process)

華為日本研究所的面試節奏通常非常緊湊（通常在 1 至 1.5 個月內完成），標準流程包含以下五個關卡：

```
+---------------+      +---------------+      +---------------+      +---------------+      +---------------+
|  1. 履歷篩選  | ---> |  2. 線上機考  | ---> | 3. 技術面試*2 | ---> | 4. 部門主管面 | ---> | 5. HR與談薪資 |
+---------------+      +---------------+      +---------------+      +---------------+      +---------------+
```

### 關卡一：履歷篩選 (Resume Screening)
*   獵頭送件後，華為品川研究所的相機團隊主管（Hiring Manager）會進行初步評估。
*   由於您的履歷已經完美融入「可變光圈對焦自動化評估」、「AF Comparator 異常診斷解決方案」，與他們的產品線高度吻合，這一關通常能快速通過。

### 關卡二：線上機考 (Online Coding Test)
*   **平台：** 通常使用牛客網（NowCoder）或華為內部系統。
*   **題型：** 3 題演算法題，總分 600 分（配分通常為 100/200/300）。
    *   *第 1 題 (簡單/中等)：* 陣列、雙指標、滑動窗口。
    *   *第 2 題 (中等)：* 深度優先/廣度優先搜尋 (DFS/BFS)、圖論、字串解析。
    *   *第 3 題 (中等/困難)：* 動態規劃 (DP)、複雜狀態機模擬。
*   **通過門檻：** 建議至少拿到 **150 分以上**（通常完成第一題，且第二題通過部分測試測資即可進入下一階段；若拿滿分會極具優勢）。

### 關卡三：技術面試兩輪 (Technical Interviews - 一面與二面)
*   **面試官：** 主要是華為品川研究所的資深相機研發工程師或技術專家（通常為中籍，可使用中文或英文進行）。
*   **核心內容：**
    1.  **機考代碼複審 (Code Review)：** 面試官會打開您之前寫的機考程式碼，問您當時的設計思路，以及「如果時間更充裕，你要怎麼優化時間/空間複雜度？」。
    2.  **專案深度探討：** 要求您畫出或詳細口述您最自豪的專案架構（例如 **AF Comparator** 的資料流、多執行緒如何避免死鎖）。
    3.  **手寫程式題 (Live Coding)：** 現場在共享白板（如 CoderPad）上寫程式。通常是 LeetCode Medium 難度，或是與影像/對焦直接相關的基礎實作（例如：拋物線極值擬合、簡單的低通濾波器）。
    4.  **相機專業與異常排查問題：** 會給予您一個實際的產品 Bug 情境（例如：「長焦切換時對焦拉風箱」或「低光源 ToF 數據異常」，問您要如何排查低底層 log 與設計補償解決方案）。

### 關卡四：業務主管面試 (Director Interview - 部長面/三面)
*   **面試官：** 品川相機研發部門的部長或所長。
*   **核心內容：**
    *   偏向高維度的產品解決方案討論。面試官可能會問：「如果未來相機要引入新的 Sensor 架構，你認為自動對焦演算法最大的瓶頸會在哪裡？你要如何設計驗證架構？」
    *   評估您的技術視野、軟實力、人格特質是否適合華為的節奏。

### 關卡五：HR 面試與談薪 (HR & Offer Discussion)
*   確認個人轉職動機、海外搬遷意願、家庭狀況。
*   討論薪資結構（底薪 + 績效年終）。

## 2. 前輩分享的「華為品川相機團隊」工作氛圍

1.  **國際化與溝通語言：**
    *   品川研究所是智慧終端（手機、穿戴裝置）的相機技術核心。團隊中有很多日本本土的相機大廠（Sony、Olympus 等）跳槽過來的資深光學與影像專家。
    *   日常技術溝通中，英文非常實用；但在專案協調、匯報主管時，中文佔有極大優勢。具備日語能力（如您的 N2）在與日本本土供應商（如 Sony 晶片廠）對接時是不可或缺的加分項。
2.  **工作步調與薪資福利：**
    *   **薪資極具競爭力：** 華為在日本給予的 Base 薪資通常高於日本傳統大手企業，且 Relocation Package（搬遷補貼、簽證贊助）相當完整。
    *   **步調較快：** 相比傳統日商較為嚴謹緩慢的節奏，華為屬於高效率、快速迭代的產品導向文化，重視「直接解決產品問題的解決方案」。

## 3. 下階段具體準備行動 (Action Plan)

1.  **刷題準備 (LeetCode)：**
    *   建議下週開始，每天練習 1-2 題 LeetCode Medium。重點複習：**BFS/DFS（圖與網格搜尋）**、**滑動窗口 (Sliding Window)**、以及**字串/Protobuf 解析**相關題型（這對機考第一、二題非常有幫助）。
2.  **機考練習：**
    *   您可以上「牛客網」練習華為的機考歷屆試題，熟悉其輸入輸出（I/O）的讀取格式（華為機考通常需要自行處理 `cin` / `sys.stdin` 讀入資料，與 LeetCode 僅寫 class method 不同）。
3.  **專案解說練習：**
    *   準備好如何口述 **AF Comparator** 的前後端架構與效能優化（ThreadPoolExecutor）。
    *   熟記您目前處理的 **Variable Aperture (可變光圈) Focus Peak Shift** 的測試數據處理流（adb -> pull log -> python parse -> curve fitting）。
